# @console-inline/service

Automatically sends all console output to Neovim as virtual text.

## Usage

### Node.js

```js
import "@console-inline/service";
console.log("Hello from Node!");
```

### React/Browser

```js
import "@console-inline/service";
console.log("Hello from browser!");
```
